YOLO Network List

COCO
15 Available Networks

The most popular and largest generic, optical object detection datasets available. Consists of 80 (object) and 91 ("stuff") categories

VOC
2 Available Networks

Visual Objects in Context (VOC) is an early generic object detection dataset. The dataset contains 20 classes in both the 2007, 2010, and 2012 versions.

KITTI
1 Available Network

A popular dataset that contains RGB dash-cam footage, corresponding depth input, and other sensor information for neural network-powered autonomous driving/detection.
